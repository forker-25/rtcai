# rtcai — бесплатный ИИ-помощник в терминале

Использует Groq API (бесплатно: 14.4k запросов/день) и Llama 3.3 70B.

## Информация и возможности
- Диалог с сохранением контекста (`ai long`)
- Диалог без сохранения контекста(`ai`)
- Выполнение команд с подтверждением
- Создание файлов с подтверждением
- Цветной вывод, спиннеры, тайминги

## Быстрый старт
```bash
git clone https://github.com/forker-25/rtcai
cd rtcai
pip install -r requirements.txt
python install.py
# Добавить GROQ_API_KEY в ~/.rtcai/.env
```
